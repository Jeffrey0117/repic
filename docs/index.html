Features

Everything you need,
nothing you don't 你需要的全都有，
不需要的一個都沒有

RePic combines the speed of a native viewer with the power of web integration. No bloat, no subscriptions, no cloud dependency. RePic 結合了原生瀏覽器的速度與網頁整合的強大功能。不臃腫、不訂閱、不依賴雲端。

Blazing fast browsing極速瀏覽

Virtual scrolling and preloaded thumbnails let you fly through thousands of images without a hiccup. 虛擬捲動 + 縮圖預載，幾千張圖滑起來跟原生一樣順暢。

Web albums — 0 KB網頁相簿 — 0 KB

Drag images from any website into virtual albums. No downloads, no disk usage — true zero space. Hotlink protection bypassed automatically. 從任何網站拖曳圖片到虛擬相簿。不下載、不佔空間 — 真正的零空間收藏。自動繞過防盜連。

.repic files.repic 虛擬圖片

Pin albums to disk as tiny .repic shortcut files (<1 KB each). Visible in File Explorer, shareable, backupable. Or keep them virtual — your choice. 將相簿釘選到磁碟，每張圖變成不到 1 KB 的 .repic 捷徑檔案。在檔案總管可見、可分享、可備份。或保持虛擬 — 由你決定。

Drag to share拖曳分享

Drag any image out of RePic straight into Discord, LINE, social media, or any other app. 把圖片直接從 RePic 拖到 Discord、LINE 或任何應用程式就能分享。

Non-destructive editing非破壞性編輯

Crop, annotate, and mosaic your images. All edits are saved separately — originals stay untouched. 裁切、標註、馬賽克，所有編輯獨立儲存，原圖完全不受影響。

Paste from clipboard剪貼簿貼上

Press Ctrl+V to instantly import screenshots, copied images, or image URLs. Zero friction. 按 Ctrl+V 就能直接匯入截圖、複製的圖片或圖片網址，零摩擦力。

.repic format

Two modes, one album 兩種模式，同一個相簿

Virtual albums live in memory — 0 KB on disk. When you want real files, pin an album to create .repic shortcut files you can see in File Explorer, share, and back up. Each file is a tiny JSON with just a URL and crop parameters — no pixel data. 虛擬相簿存在記憶體中 — 磁碟佔用 0 KB。想要實體檔案時，釘選相簿就會產生 .repic 捷徑檔，在檔案總管可見、可分享、可備份。每個檔案只是一個 JSON，只有 URL 和裁切參數，完全沒有像素資料。
